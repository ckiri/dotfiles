# .dotfiles WSL branch

The WSL branch of the `.dotfiles` repository contains configuration files for
a efficient workflow inside WSL.

This dotfiles repository contains following config files:
* dirs -- sets variables for system/config paths
* nvim -- neovim, texteditor config with LSP integration
* tmux -- terminal multiplexer
* vim -- text editor config, held minimalist
* zsh -- the best shell ever, contains scripts for ease of use
